路由实现 url 转发：

要实现路由，就必须进行 url 重写（也称 pathinfo 模式），而 url 重写是建立在 url 转发的基础上的，url 转发由 apache 的 mod_rewrite.so 模块（较新版本的 apache 默认开启该模块）提供，在网站根目录新建 .htaccess 文件，粘贴如下代码：

<IfModule mod_rewrite.c> Options +FollowSymlinks -Multiviews RewriteEngine On RewriteCond %{REQUEST_FILENAME} !-d RewriteCond %{REQUEST_FILENAME} !-f RewriteRule ^(.*)$ index.php/$1 [QSA,PT,L] </IfModule>

OK，这样就在该目录下开启了 url 转发，该目录的所有请求都会被转发到 index.php 文件，之后就可以通过解析 url 来实现路由了。

publicModel 实现 获取子类 model 的类名：

当子类未定义构造函数时，php 就会调用父类的构造函数，此时在父类的构造函数内通过 basename(get_class($this)) 就可以获取到子类的类名进行操作。
