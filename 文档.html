公共函数

支持全局公共函数、模块公共函数，需在 config 文件（全局、模块都可以）内配置，将文件名加入 'publicFuncFile' 即可。

全局公共函数：

路径：/publicFunc/文件名

作用于整个应用。

模块公共函数：

路径：/app/模块名/文件名.php

只在当前模块内生效，若与全局公共函数文件同名，会覆盖掉全局公共函数文件。

配置文件

支持全局配置文件、模块配置文件。

全局配置文件：

路径：/config/config.php

作用于整个应用。

模块配置文件：

路径：/app/模块名/config.php

用于为模块进行个性化配置，该文件会覆盖全局配置部分参数。格式固定：

<?php $模块名 = [ // 变量名与模块名保持一致 ]; // 覆盖全局 config 的部分参数 foreach($模块名 as $k => $v) { $config[$k] = $模块名[$k]; }
